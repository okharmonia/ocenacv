## 0. ПСИХОЛОГИЯ ВОРОНКИ (что должно происходить в голове)

```
        «Ciekawe, o co chodzi...»       ← интрига
                   ↓
        «No dobra, wrzucę to CV»         ← низкий порог (FOMO: "sprawdź za darmo")
                   ↓
        «Kurczę, 34/100?!»               ← шок / сомнение в своём CV
                   ↓
        «Racja, piszą konkretnie...»     ← доверие (замечания точные)
                   ↓
        «To co mam z tym zrobić?»        ← поиск решения
                   ↓
        «gen-cv.pl — sprawdzę»            ← действие
```

**Главный принцип:** каждая строка на странице либо удерживает, либо продвигает вниз по воронке.
Ни одной строчки без задачи.

```

  linear-gradient(180deg, #0F172A 0%, #1E293B 100%)

  SVG белая волна, создаёт ощущение «перетекания»
```

### Цвета ATS-score

```
Score 0-35   → #EF4444 (красный)   — «Твоё CV — список покупок, не резюме»
Score 36-55  → #F97316 (янтарный)  — «Ну такое... рекрутер зевнёт»
Score 56-75  → #EAB308 (жёлтый)    — «Почти норм, но не выделяется»
Score 76-100 → #10B981 (зелёный)   — «Отлично, но всегда можно лучше»
```

### Правило цвета

- На странице **НЕ БОЛЬШЕ 2 акцентных цветов одновременно**
- До результата: тёмный Navy + чистый белый + один оранжевый CTA
- После результата: красное/жёлтое число + зелёный CTA на gen-cv
- **Серый используется ТОЛЬКО для подписей и разделителей**, никогда для CTA
- **Никаких радуг. Дисциплина цвета = доверие.**

## 5. ВИЗУАЛЬНЫЕ ЭЛЕМЕНТЫ

### 5.1 Иконки

*Никаких 3D, никаких градиентных иконок с тенями. Только чистые, монохромные, контурные иконки в стиле Phosphor Icons или Lucide. Толщина линии: 1.5px.*

```
Upload:     <иконка загрузки (стрелка вверх + поднос)>
AI:         <иконка искры / молнии / мозга (простой контур)>
Score:      <иконка документа с галочкой>
Warning:    <треугольник с восклицательным знаком>
Success:    <круглая галочка>
Arrow:      <стрелка вправо для CTA>
Lock:       <замок — для trust-блоков>
```

### 5.2 Иллюстрации

*Hero-секция: НЕ использовать стоковые фото людей в костюмах. Вместо этого:*

```
Hero: абстрактная геометрическая композиция справа.
      — Тёмно-синий фон (#0F172A)
      — Справа: стилизованный «документ» из геометрических линий,
        из которого «вылетают» оранжевые акцентные точки/частицы
      — Слева: текст (заголовок + саблайн + CTA)
      — Между ними: воздух (негативное пространство)
```

*Если нужен визуал для секции «Jak to działa»:*
```
— Простые контурные иллюстрации (стиль unDraw но в цветах бренда)
— НЕ перегружать, максимум 3 штуки
— Высота иконки/иллюстрации: 64-80px
```

### 5.3 ДЕКОРАТИВНЫЕ ЭЛЕМЕНТЫ

```
— Тонкие линии-разделители (1px, #E2E8F0)
— Лёгкие градиентные пятна на фоне секций (opacity 0.03-0.06)

## 7. АНИМАЦИИ И МИКРО-ВЗАИМОДЕЙСТВИЯ

### Принцип

*Анимация не украшает — она направляет внимание. Каждое движение должно что-то значить. Никаких «просто красиво».*

### Конкретные анимации

```
1. Hero — fade-in
   Заголовок, саблайн, иконки: появляются с задержкой
   100ms между элементами, fade-in up (20px), duration 400ms

2. Карточки «Jak to działa»
   Появляются по очереди при скролле (Intersection Observer)
   Fade-in up, stagger 150ms

3. Upload-зона — пульсация рамки
   Лёгкая пульсация dashed-рамки (opacity 0.6 → 1.0, цикл 3s)
   Привлекает внимание: «здесь действие»

4. Загрузка (AI thinking)
   — Иконка документа → иконка молнии (morphing или fade-переход)
   — Прогресс-бар: тонкая полоса под заголовком, заполняется
   — Текст меняется: «Analizuję CV...» → «Szukam słabych punktów...»
     → «Oceniam formatowanie...» → «Gotowe!»
     (каждый этап 2-3 секунды, создаёт ощущение «работает серьёзно»)

5. Результат — reveal
   — Круглая диаграмма: счётчик от 0 до score (1.5s ease-out)
   — Полоса: заполняется слева направо (0.8s после цифры)
   — Приговор: появляется с эффектом печатной машинки
     «Rekruter odrzuci to CV w 6 sekund»
     (по 1 букве, 30ms каждая — быстро, но читаемо)
   — 5 карточек с замечаниями: fade-in up, stagger 200ms

6. CTA на gen-cv.pl
   — Пульсирующее свечение кнопки: box-shadow с оранжевым,
     пульсация 2s бесконечно (привлекает взгляд после чтения замечаний)
   — При наведении: scale(1.05), свечение усиливается
```

### Что НЕЛЬЗЯ

```
— Параллакс-эффекты (замедляют, раздражают на мобильных)
— Автовоспроизведение видео
— Всплывающие окна (popup — смерть доверия)
— Бесконечный скролл с анимациями (lighthouse сожрёт)
— Слишком быстрые анимации (<200ms) — дёшево выглядит
— Слишком медленные (>600ms) — раздражает
```

   bezpiecznego połączenia»

Footer:
  «Polityka prywatności» (ссылка, даже если страницы ещё нет —
  placeholder, который говорит «мы серьёзные»)
```

## 10. SEO-ТРЕБОВАНИЯ (для дизайнера: не про красоту, а про разметку)

*Это не «потом добавим». Это в дизайне сразу.*

```
— h1: ровно ОДИН на странице (hero заголовок)
— h2: каждая смысловая секция (Jak to działa, Przykład, Dlaczego, Wynik)
— Все изображения/иконки: с alt-текстом
— Цвета не несут смысловой нагрузки (не «нажми красную кнопку»,
  а «нажми кнопку z napisem Sprawdź CV»)
— Контент читается без JS (graceful degradation)
— Кнопки — это <button> или <a>, не div с onclick
— Контраст текста: минимум 4.5:1 для body, 3:1 для крупного
```

## 11. ТЕХНИЧЕСКИЕ ОГРАНИЧЕНИЯ (влиюящие на дизайн)

```
— Шрифты: только Inter (Google Fonts), без дополнительной загрузки
— Никаких видео-фонов (трафик + lighthouse)
— SVG-иллюстрации (не PNG, не WebP для графики)
— Иконки: SVG-спрайт или inline (не картинки)
— Максимальный вес страницы: <200 KB (без учёта загружаемого CV)
```

## 12. ЧЕК-ЛИСТ ДЛЯ ДИЗАЙНЕРА

```
✅ Hero захватывает за 2 секунды
✅ Кнопка CTA видна без скролла (above the fold)
✅ Upload-зона не похожа на рекламный баннер
✅ Цвета не отвлекают от действия
✅ Before/After — честный пример (не выдуманный)
✅ На мобильном кнопка закреплена или легкодоступна
✅ Анимации усиливают понимание, а не просто «красиво»
✅ Контраст текста проходит accessibility
✅ Форма работает без JS? (хотя бы видна)
✅ Понятно что произойдёт после нажатия на кнопку?
✅ Ясно, что CV не сохраняется?
✅ Ссылка на gen-cv.pl логично встроена (не выглядит как реклама)?
```

## 13. ПРИМЕРЫ ДЛЯ ВДОХНОВЕНИЯ

*Не копировать. Изучить ритм, плотность, воздух.*

```
— cvroast.com        — агрессивный tone, минимализм, score + 5 roasts
— roasted.cv         — тёмная тема, gamified
— resumly.ai/resume-roast — чистая форма, хороший before/after
— enhancv.com        — польский конкурент? посмотреть UI
```

### Чего НЕ делать

```
❌ Длинные тексты-простыни (пользователь пришёл проверить, не читать)
❌ Карусели (никто не листает)
❌ Popup «zapisz się do newslettera» (убивает доверие)
❌ Счётчики «осталось 3 места» (false urgency, поляки чуют)
❌ Стоковые фото улыбающихся людей в костюмах (2020 звонит)
```
